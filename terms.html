Terms and Conditions


In order to rent a motor, the following requirements must be met:

1. Security Deposit:
A security deposit of ₱3,000 is required for daily rentals, ₱5,000 for weekly rentals, and ₱15,000 for monthly rentals.

2. Valid Government-issued IDs:
Two valid government-issued IDs are required for the rental process. Acceptable forms of identification include UMID (Unified Multi-Purpose ID), SSS (Social Security System) ID, driver's license, passport, and other similar government-issued IDs.

3. Surrender of Valid ID:
One valid ID will need to be surrendered either upon picking up the motor or when it is delivered to you. Please ensure that you meet these requirements in order to proceed with the motor rental.
